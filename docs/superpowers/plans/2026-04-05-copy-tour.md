> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a "Copy" button to the admin tour index that duplicates a tour (with all steps) as a draft via a `TourCopier` service.

**Architecture:** A `TourCopier` service handles duplication logic (tour + steps) in a transaction. A new `copy` member route on tours calls the service from the controller. The copy button is a `button_to` in the tour partial.

**Tech Stack:** Ruby on Rails engine, RSpec, FactoryBot

- [ ] **Step 3: Implement the service**

Create `app/services/onboard_on_rails/tour_copier.rb`:

```ruby
module OnboardOnRails
  class TourCopier
    def self.call(original)
      new(original).call
    end

    def initialize(original)
      @original = original
    end

    def call
      ActiveRecord::Base.transaction do
        new_tour = @original.dup
        new_tour.name = "#{@original.name} #{I18n.t('onboard_on_rails.admin.tours.copy_suffix')}"
        new_tour.status = "draft"
        new_tour.save!

        @original.steps.each do |step|
          new_step = step.dup
          new_step.tour = new_tour
          new_step.save!
        end

        new_tour
      end
    end
  end
end
```
